# How-to-run-TLauncher-On-Linux

# This post is for those who had problems installing T-Launcher.


---


# Setup 1
Download 

    Go to (https://tlauncher.org/)
    
    Create OR Signup your TLauncher Account
    
    Click Download, Click Linux And Download tlauncher-linux-installer.deb 

---


# Setup 2 : Permision And Install TLauncher

    chmod +x tlauncher.deb && sudo apt install ./tlauncher-linux-installer.deb

---

# Setup 3 : How To Play?
    
    Click And Move To apps And click TLauncher App
